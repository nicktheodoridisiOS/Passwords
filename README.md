### About Application
Passwords is an iOS mobile app developed with Swift programming language. In this app you have the option through some fields to generate passwords and create your own showing you how strong it is and you can save them in your library.

### Build Your Password Library
<img width="1200" alt="library" src="https://github.com/nicktheodoridisiOS/Passwords/assets/122683142/13cd7422-115a-4622-b8fb-ff89b335ced1">

```Swift
import Foundation
import SwiftData
import SwiftUI

@Model
class Password{
    var name: String
    var color: String
    
    init(name: String, color: String) {
        self.name = name
        self.color = color
    }
}
```
